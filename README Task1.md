# News Topic Classifier Using BERT

## Objective
Fine-tune BERT (bert-base-uncased) on the AG News dataset to classify news headlines into four categories:
- World
- Sports
- Business
- Sci/Tech

## Dataset
AG News Dataset from Hugging Face.

## Technologies Used
- Python
- Transformers
- BERT
- Hugging Face Datasets
- Scikit-learn
- Gradio

## Evaluation Metrics
- Accuracy
- F1 Score

## Results
- Accuracy: ~94%
- F1 Score: ~94%

## Run

pip install -r requirements.txt

python news_classifier.py
